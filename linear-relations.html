Linear Relations - Multiple Choice Practice

1. Using the relation y = 2x + 5:
i) What is y when x = 3?
ii) What is x when y = 25?

2. Using the relation y = -3x + 4:
i) What is y when x = 2?
ii) What is x when y = 7?

3. Using the relation y = 4x - 3:
i) What is y when x = 2?
ii) What is x when y = 17?

4. Using the relation y = -2x + 8:
i) What is y when x = 5?
ii) What is x when y = -4?

5. Using the relation y = ½x + 10:
i) What is y when x = 6?
ii) What is x when y = 16?

6. Using the relation y = -x + 7:
i) What is y when x = 4?
ii) What is x when y = 3?

7. Using the relation y = 3x - 5:
i) What is y when x = 0?
ii) What is x when y = 10?

8. Using the relation y = 10 - 2x:
i) What is y when x = 3?
ii) What is x when y = 4?
